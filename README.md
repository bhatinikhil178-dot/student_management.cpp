# Student Management System

A simple C++ mini project for managing student records.

## Features
- Add Student
- Display Students
- Search Student
- Delete Student
- Menu Driven Interface

## Technologies Used
- C++
- Object-Oriented Programming (OOP)

## Concepts Used
- Classes and Objects
- Functions
- Arrays
- Loops
- Conditional Statements

## How to Run
1. Compile the code using any C++ compiler.
2. Run the executable file.
3. Choose options from the menu.
